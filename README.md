# 🧮 CoreCalc

CoreCalc is a modern, responsive, and user-friendly calculator built using **HTML**, **CSS**, and **JavaScript**. 
It provides a clean interface with smooth interactions, making basic arithmetic calculations quick and efficient.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear display
- ⌫ Delete last input
- 📜 Calculation history
- 💡 Modern glassmorphism UI
- 🌟 Glowing border effects
- 📱 Responsive design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

1. Clone the repository.

2. Open the project folder.

3. Launch `CoreCalc.html` in your web browser.

No installation or dependencies are required.
